## Installed Resources

The main view shows all plugins (Spigot/Paper) or mods (Fabric) currently installed on your server:

| Information | Description |
|-------------|-------------|
| **Name** | The plugin/mod name |
| **Version** | Currently installed version |
| **Description** | Brief description of functionality |
| **Author** | The developer(s) |
| **Status** | Whether it's enabled or disabled |

### Managing Installed Resources

- **Enable/Disable**: Toggle plugins on or off (requires server restart)
- **Delete**: Remove the plugin/mod file from the server
- **Update**: If an update is available, install the latest version

## Marketplace

Browse and install plugins/mods directly from popular repositories:

### Supported Providers

::tabs{variant="card"}
::div{label="Modrinth" icon="simple-icons:modrinth"}
Browse the Modrinth catalog for both plugins and mods. Modrinth is open-source and has a large collection of quality resources.

**No API key required** - Works out of the box!
::

::div{label="Spigot" icon="lucide:box"}
Access the SpigotMC resource catalog with thousands of plugins.

**Requires API key** - You'll need to configure your Spigot API key in settings.
::

::div{label="CurseForge" icon="simple-icons:curseforge"}
Browse CurseForge's extensive mod collection for Fabric and Forge servers.

**Requires API key** - You'll need to configure your CurseForge API key in settings.
::
::

### Installing from Marketplace

1. Switch to the **Store** tab
2. Search for the plugin/mod you want
3. Click on a result to view details
4. Select the version compatible with your server
5. Click **Install**

MCDash will automatically download and install the resource to the correct folder.
